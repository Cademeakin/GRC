Overwatch

This game is incredible. Overwatch has done something that no other FPS has done in years: be truly engaging and fun. The classes are all fun to use, each with their own personalities. The game modes are hype-filled thrill rides, the gameplay is balanced to where people who don't like shooters can enjoy, the maps are all fun to play on, and almost everything about this game is absolutely amazing. It's easily one of Blizzard's best games in years, and while Overwatch isn't the best shooter i've ever played, but I can see myself playing this game for a long time coming. It's one of the best games i've played this year, and no other shooter this year will come close to what this game has accomplished.
